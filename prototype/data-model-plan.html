2四大护航 · 数据模型实现方案
每条护航包含：业务定义 → 数据源 → 关键特征 → 模型算法 → 三级阈值 → 输出与派单。所有阈值均为初始建议值，最终以校方审定结果为准。
2.1　行为护航 · 归巢行动
通过宿舍门禁 + 智能门锁数据，识别晚归、未归与作息异常，保障在校学生人身安全
规则模型 + 时序统计
数据源
宿舍门禁系统 宿舍智能门锁 校园门禁系统 请假/外出审批
数据频率：实时（事件流），T+0 入仓，秒级触发预警
关键特征（Feature）
本次归寝时间 本周晚归次数 本月晚归次数 本学期晚归累计 连续晚归天数 连续未归天数 归寝时间方差 是否处于排除时段
模型与判定逻辑
晚归判定：当日 22:30 之后未刷宿舍门锁/门禁入寝
未归判定：当晚至次日 06:00 仍无归寝记录
排除时段：寒假 / 暑假 / 法定节假日（默认排除），周末（可选）
排除身份：休学 / 退学 / 毕业离校 / 校外实习（白名单）
输出与派单
· 一级 → 辅导员（48h 反馈）
· 二级 → 辅导员 + 学院副书记（24h 反馈）
· 三级 → 辅导员 + 副书记 + 学院党委书记（12h 反馈，自动通知家长备案）
三级阈值规则（建议初始值）
等级触发条件推送对象反馈时限
一级 · 黄单次晚归 22:30 ~ 23:59辅导员48 小时
二级 · 橙单次未归 或 连续晚归 ≥ 3 天辅导员 + 副书记24 小时
三级 · 红连续未归 ≥ 2 天 或 月累计晚归 ≥ 10 次辅导员 + 副书记 + 党委书记12 小时
2.2　学业护航 · 登高计划
基于课堂考勤、成绩、作业提交等数据，识别学业困难与挂科风险，及时提供学业帮扶
规则模型 + 梯度提升回归 (GBDT)
数据源
课堂考勤系统 教务成绩系统 作业 / 实验提交平台 期中/期末考试 补考重修记录 学业咨询记录
数据频率：日 (考勤) / 周 (作业) / 学期 (成绩) 增量同步
关键特征
周旷课次数 连续旷课天数 月累计缺勤 本学期 GPA GPA 同期下降幅度 挂科门数 作业完成率 学院/班级排名 期中模拟成绩
模型与判定逻辑
规则部分：对出勤、挂科、GPA 等强相关指标设置硬规则触发
AI 部分：引入 GBDT 模型预测"期末挂科概率"，输入 30+ 维历史特征
计算频率：出勤日更，成绩 GPA 周更，挂科风险预测周更
排除身份：休学复学适应期（白名单 1 学期）
输出与派单
· 一级 → 辅导员 + 任课教师提示
· 二级 → 辅导员 + 学业导师 + 副书记
· 三级 → 辅导员 + 学院教学副院长（启动学业帮扶计划，通知家长）
三级阈值规则（建议初始值）
等级出勤异常成绩 / 学业综合判定（满足任一）
一级 · 黄周旷课 ≥ 3 次单门期中 < 60 分挂科风险 > 40%
二级 · 橙连续旷课 ≥ 3 天 或 周旷课 ≥ 5 次挂科 ≥ 2 门 或 GPA 同期下降 ≥ 0.5挂科风险 > 65%
三级 · 红连续旷课 ≥ 5 天 或 周旷课 ≥ 7 次挂科 ≥ 3 门 或 排名同期下降 ≥ 30%挂科风险 > 85%（满足学业警告条件）
2.3　心理护航 · 晨曦计划
基于 NLP 语义分析，从校园论坛/邮件/聊天文本中识别抑郁、焦虑、危机倾向，三级预警自动派单至心理咨询师
NLP 语义模型 + 情感倾向 + 量表评估
数据源（合规边界内）
校园 BBS / 论坛公开帖 企业邮箱（@csu.edu.cn） 官方聊天 / 工单 公开聊天群 心理量表 PHQ-9 / GAD-7 心理咨询记录
合规说明：仅在校内合法采集范围内、命中关键词后才启用人工复核；不接入个人微信/QQ 私聊；遵循《中南大学学生数据安全管理办法》。
关键特征
7 日抑郁词触发频次 7 日焦虑词触发频次 高危词命中次数 情感倾向得分 (-1~1) PHQ-9 / GAD-7 量表分 关键词共现强度 触发时段集中度 社交隔离度
模型与判定逻辑
NLP 引擎：基于私有化部署 DeepSeek + 心理领域微调词库，对全文做分句 → 关键词匹配 → 情感倾向打分 → 上下文判定
词库分级：抑郁词 / 焦虑词 / 高危词三类，支持隐喻、反讽场景
计算频率：实时流式计算（命中即触发）
人工复核：所有命中均由心理中心专员二次确认，避免误伤
输出与派单
· 一级 → 辅导员（48h 内电话/线下关怀）
· 二级 → 心理咨询师 + 辅导员（24h 内 1v1 咨询）
· 三级 → 5 分钟内派单：危机干预小组 + 辅导员 + 保卫处，并通知家属
三级阈值规则（建议初始值）
等级关键词集合（示例）触发条件派单
一级 · 黄
抑郁倾向		emo / 难受 / 没意思 / 没意义 / 不想 / 撑不下去 / 累 / 失眠7 日内 ≥ 3 次辅导员（48h）
二级 · 橙
焦虑/重度		绝望 / 撑不下去 / 想躲起来 / 无意义 / 焦虑 / 崩溃 / 孤独7 日内 ≥ 5 次
或 单条多词共现		心理咨询师 + 辅导员（24h）
三级 · 红
危机倾向		想不开 / 解脱 / 跳 / 自杀 / 不想活 / 一了百了 / 遗书单次命中即触发
（含隐喻 / 反讽）		危机干预小组 + 辅导员 + 保卫处（5min）
2.4　经济护航 · 春雨行动
基于一卡通消费数据，识别隐性贫困与异常消费，做到"主动发现 · 精准帮扶 · 不伤自尊"
时序异常检测 + 同群对比
数据源
校园一卡通（食堂/超市/水电） 奖助学金系统 学费缴纳系统 勤工助学 家庭经济困难认定
数据频率：T+1 同步消费明细，月级别校验奖助/学费
关键特征
日均消费金额 近 7 日 vs 前 30 日均值 单笔最大消费 食堂消费天数占比 餐次完整性（早/午/晚） 连续低/零消费天数 同学院同年级中位数对比 奖助 + 勤工补贴情况
模型与判定逻辑
规则部分：消费骤升/骤降/低消费/零消费四类硬规则
统计部分：z-score 异常检测 + 同年级同学院消费水位对比
计算频率：每日凌晨例行任务（T+1 日报，T+7 周报）
排除身份：请假离校 / 校外实习 / 食堂未开放期间
输出与派单
· 一级 → 辅导员关注（不上门、不主动说明）
· 二级 → 辅导员私下沟通 + 提示资助中心
· 三级 → 资助中心介入，启动隐性帮扶（餐补/补助）
三级阈值规则（建议初始值）
等级消费骤降低/零消费消费骤升
一级 · 黄近 7 日 vs 前 30 日 ↓ 30%~50%日消费 < ¥10 持续 ≥ 3 天日均 ↑ 50%~100%
二级 · 橙近 7 日 vs 前 30 日 ↓ 50%~80%日消费 < ¥10 持续 ≥ 7 天日均 ↑ > 100% 或 单笔 > ¥500
三级 · 红近 7 日 vs 前 30 日 ↓ > 80%连续 ≥ 3 天零消费单日 > ¥1000 或 单笔 > ¥800
3通用规则与策略
3.1　预警自动升级
一级 → 二级
超过 48 小时 未反馈 或 同一类型连续触发 ≥ 3 次
二级 → 三级
超过 24 小时 未反馈 或 同一类型连续触发 ≥ 5 次
催办策略：到达反馈时限 50% 时首次催办，80% 时二次催办（企业微信 + 短信），超时通知上级领导。
3.2　白名单机制（避免误伤）
类别典型场景申请人有效期
导师 / 项目类实验室赶项目（需导师确认）、竞赛集训导师 / 学院1 ~ 3 个月，可续期
身份类休学复学适应期、交换生（出境）、校外实习辅导员1 学期 / 实习结束
医疗类住院 / 长期病假 / 心理咨询治疗中辅导员 + 校医院按康复进度
所有白名单需走"申请 → 学院审批 → 学工部备案"三级审批流程，过期自动失效。
3.3　全局排除规则
按学生状态排除
休学 · 退学 · 毕业离校（默认）；交换生（出境）、校外实习（按需勾选）
按时段排除
寒假 · 暑假 · 五一 · 国庆 · 元旦（默认）；周末（行为护航可选排除）
4数据合规与隐私保护
最小必要原则
仅采集与预警直接相关的字段；学生个人微信/QQ 私聊一律不接入；无关字段（消费品类细节、聊天人物关系）在脱敏层裁剪。
分级访问控制
辅导员仅可查看本班学生预警；副书记/书记按学院；心理咨询师仅看心理类；学校管理员看汇总指标，不下钻到个人详情。
操作留痕审计
所有学生详情查看、画像查询、规则修改全部留痕，可追溯到人；日志保留 ≥ 3 年；学工部按月抽查。
私有化部署
DeepSeek 大模型本地部署在校内 IDC，文本不出校；与外部任何 API 无连接。
人工复核机制
心理类、危机类预警必须经心理中心专员人工复核后方可推送给辅导员；避免 NLP 误判带来二次伤害。
学生知情权
学生可在个人门户查看自身被命中的预警类型、命中次数（不含具体词条原文），并可申诉/申请白名单。
5模型评估指标（KPI）
护航核心指标目标值评估方式
行为护航预警精准率 / 漏报率精准 ≥ 92%   漏报 < 3%抽样人工核验（每月 100 例）
学业护航挂科预测 AUC / 召回率AUC ≥ 0.85   召回 ≥ 80%学期末成绩回溯验证
心理护航关键词识别准确率 / 误报率准确 ≥ 88%   误报 < 5%心理中心专员双盲复核
经济护航主动发现率 / 帮扶接受率主动 ≥ 75%   接受 ≥ 60%对比已认定贫困库 + 后续跟踪
整体目标：预警闭环率 ≥ 95%、平均反馈时长 < 24h、年度严重事件同比下降 ≥ 30%。
6校方确认清单 请逐项勾选确认 / 调整
6.1　数据源与采集范围
确认 4 大护航接入数据源边界
行为：宿舍门禁 + 智能门锁；学业：教务/考勤/作业；心理：校园论坛/邮件/官方聊天 + PHQ-9/GAD-7；经济：一卡通 + 奖助学金 + 学费
心理护航：明确"个人微信/QQ 私聊不接入"
仅接入校内合法采集场景（论坛/官方邮箱/官方聊天/工单/草稿箱），所有命中由心理中心专员人工复核后再派单。
6.2　预警阈值（核心，请重点确认）
行为护航 · 晚归判定时间 22:30 / 未归判定 06:00
是否调整时间？是否周末同样适用？
学业护航 · 出勤异常 / 挂科 / GPA 下滑三类阈值
三级触发是否对应学校"学业警告"标准？建议三级 = 学业警告。
心理护航 · 三级"高危词"集合及单次命中即触发
默认词集：想不开/解脱/跳/自杀/不想活/一了百了/遗书。是否补充其他校内常见隐喻？
经济护航 · 隐性贫困识别（"不上门、不主动说明"原则）
一二级仅辅导员关注，不约谈不公示；三级才由资助中心通过校内餐补/资助通道隐性补给。
6.3　预警派单与响应时限
三级响应时限：行为/学业 12h，心理 5min（危机干预小组）
心理三级是否启动校危机干预流程？保卫处是否同步响应？
自动升级规则：未反馈超时升级为更高一级
一级 → 二级（48h），二级 → 三级（24h）；催办：50%/80% 时点，企业微信 + 短信。
6.4　白名单与排除规则
白名单审批流：辅导员申请 → 学院审批 → 学工部备案
是否设置最长有效期上限？过期是否自动重审？
默认排除时段：寒假/暑假/法定节假日
周末是否纳入排除？心理护航是否在节假日仍保持开启（建议保持开启）？
6.5　数据合规与权限
分级访问：辅导员/副书记/书记/咨询师/管理员的数据可见范围
学校管理员仅看汇总指标，不下钻到个人原始数据；操作留痕审计 ≥ 3 年。
学生知情权：可在个人门户查看自己被命中的类型与次数
不展示具体命中文本原文；提供白名单申诉入口。
6.6　模型 KPI 验收标准
行为/学业/心理/经济模型的精准率与召回率目标
参考 §5：行为精准 ≥ 92%、学业 AUC ≥ 0.85、心理准确 ≥ 88%、经济主动发现 ≥ 75%。
